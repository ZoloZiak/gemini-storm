# STORM Deep Research Extension

High-fidelity research reports using Gemini nested execution.

## Commands
- `/deep-research "topic"`: Start background research.
- `/deep-research:status`: Check progress.
- `/deep-research:view`: View results.

## Requirements
- Python 3.10+
- `stanford-storm` library installed.
- Valid `YOU_API_KEY` in environment.
